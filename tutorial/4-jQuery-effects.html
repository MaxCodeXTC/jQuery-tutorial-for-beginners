jQuery effects

The jQuery library provides several techniques for adding animation to a web page. These include simple, standard animations that are frequently used, and the ability to craft sophisticated custom effects.

Hide

We can easily hide any element on our page:


    $("#panel1").hide()

What happens here - our code loads up the page and then it hides panel one.

Animation time

In order to notice it, let's add argument to our hide() function, let's pass 500 and see what will happen


    $("#panel1").hide(500);

That's easy, isn't it?

Chaining

jQuery is chainable, this is awesome feature. That means that we call multiple functions on 1 selector. Like this:


    $("#panel1").hide(500).show(1000).hide(500).show(1000);

You can keep chaining it as long as it's not semicolon yet.

Fade

Another available animation is fadeIn and fadeOut


    $("#panel1").fadeIn(200)0.fadeOut(2000);

This allows us to simulate blinking:


    $("#panel1").fadeIn(200).fadeOut(200).fadeIn(200).fadeOut(200).fadeIn(200).fadeOut(200).fadeIn(200).fadeOut(200);

Slide

We can also slide our content up and down:


    $("#panel1").slideUp(500).slideDown(500).slideUp(500).slideDown(500);

Most of the animation can simply toggled, so we don't have to know the current state of it:


    $("#panel1").slideUp(500).slideToggle(500).slideToggle(500).slideDown(500);

Delays

We can also add some delays to our effects:


    $("#panel1").slideToggle(500).delay(1000).slideToggle(500);
