document.ready()

A page can't be manipulated safely until the document is "ready." jQuery detects this state of readiness for you. Code included inside $( document ).ready() will only run once the page Document Object Model (DOM) is ready for JavaScript code to execute.

Code included inside


    $( document ).ready(function() { ... })

will run once the entire page (images or iframes), not just the DOM, is ready.

So what document.ready does is saying - hey wait for my whole document to load up before I will try running any JavaScript otherwise you will end up talking js to panel 1 but panel 1 is not there yet.


    // A $( document ).ready() block.
    $( document ).ready(function() {
        console.log( "ready!" );
    })

The shorter version:


    // Shorthand for $( document ).ready()
    $(function() {
        console.log( "ready!" );
    });
MIST

Passing named function


    function readyFn( jQuery ) {
        // Code to run when the document is ready.
    }
     
    $( document ).ready( readyFn );

Using "VanillaJS" means using plain JavaScript without any additional libraries like jQuery.

In the previous webinar we were using JavaScript to manipulate HTML elements, i.e. to edit it's text we used:


    let p = document.getElementById('panel1');

jQuery makes it much easier


$("#panel1")

and instead of long pure JS setting: 


    document.getElementById('panel1').innerText = "Some text";

much shorter version:


    $("#panel1").text("Some text");

What else we can do?

Add / remove / class 


    $("#panel1").addClass("some-class")
    $("#panel1").removeClass("other-class")
    $("#panel1").toggleClass("toggle-class")

Check if element has given class:


    console.log($("#panel1").hasClass("some-class"));

Add styles


    $("#panel1").css({color:'red'})

One think we can't do this way is use dash:


    $("#panel1").css({margin-top:'20px'})

In Javascript dash means minus so this will throw an error. But we use either:


    $("#panel1").css({marginTop:'20px'})

jQuery will understand that. Or wrap it with quotes if you want to do it that way


    $("#panel1").css({'margin-top':'20px'})

In general you css syntax looks like:


    $(selector).css(propertyname, value);

You can pass more styles:


    $("#panel1").css({
        marginTop:'20px',
        color: 'red',
        backgroundColor: 'black'
    })

And even use variables 


    let design = {
        marginTop:'20px',
        color: 'red',
        backgroundColor: 'black'
    }
    
    $("#panel1").css(design)
