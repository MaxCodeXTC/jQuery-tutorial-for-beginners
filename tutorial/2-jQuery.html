What is jQuery?

jQuery is the easiset and most popular Javascript library. So if you wanna start learning we development learn jQuery. Every employer I ever worked used jQuery no matter how big or small they were.

jQuery features

  • Easy to manipulate DOM
  • Massive community
  • Widely used
  • Cross-browser support
  • Chainable

Why should you use it?

  • Simplified JavaScript
  • EventHandling
  • Lightweight
  • Animations

DOM

The Document Object Model (DOM) is a programming interface for HTML and XML documents. It represents the page so that programs can change the document structure, style, and content. The DOM represents the document as nodes and objects. That way, programming languages can connect to the page.
