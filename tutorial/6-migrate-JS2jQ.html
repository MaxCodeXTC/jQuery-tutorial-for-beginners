Migrate our last todo app JS code to jQuery

  1. Clone or download lesson files about JavaScript from this github repo.
  2. Navigate to final/ directory and open it the editor (i.e. VSCode)
  3. Add jQuery to it (in index.html
  4. Edit js/scritps.js file and replace each JavaScript function with jQuery. If you don't know how - try to google it i.e. insertAdjacentHTML jQuery

Note: If you face some issues, you can always navigate to final/todo-app-jquery folder in this tutorial and find final jQuery version of todo app to compare with your code.
